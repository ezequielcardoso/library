湖北警官学院图书馆简介

      
 湖北警官学院图书馆是学院的文献信息保障中心，为处级建制单位。 

　　         
 目前图书馆下设图书采编室、图书流通室、期刊室、网络信息中心。全馆在编人员22人，其中男9人，
        女13人，高职人员6人，中职人员10人，初级职称5人；学历层次方面有在职硕士5人，本科12人，专科
   5人；专业构成方面有图书情报专业10人，其他专业12人。 

	　 
 目前南北两个校区图书馆总面积为17488平方米。南院图书馆建筑面积4888平方米，设有图书流通书
       库和报纸、现刊、特种文献阅览室等各类阅览室3个，阅览座位314个，电子阅览室2个，电子阅览座位122个；
       学生自习室一个，设座位208个。北区新馆建筑面积12600平方米，普通阅览室三个，座位1376个，其中图
       书阅览室设座位296个，工具书阅览室设座位216个，报刊阅览室设座位864个；电子阅览室二个，校园网
       电子阅览室设209座，公安网电子阅览室设110座；学生自习室一个，设座位156个。全馆配有各类计算机497台，
       各项业务工作实现了自动化管理。 

	　
 图书馆馆藏文献总量已达600983册，其中中文图书351197册，外文图书1087册，中文期刊合订本18618册，
      外文期刊合订本81册，总计370983册,另有电子图书230000册。 

	　
 图书馆开展了图书外借、图书阅览、报刊阅览、电子阅览、情报咨询等服务。图书馆实现了文献信息的自动化检索
      与查询，组织开发了馆藏书目数据库、公安文献篇名数据库、公安期刊全文光盘数据库，购买了中国学术期刊光盘版、
      中国法律法规大全、中国法律年鉴等数据库，能为师生提供因特网浏览、光盘检索、数据库联机检索等服务，
      图书馆逐步完成了从传统型图书馆向现代化图书馆的转型。 

    　     
 图书馆历经以下几个发展阶段：从1999年9月开始拥有独立馆舍。目前是南北校区各有一图书馆，  
      总面积为17488平方米。
